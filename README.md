# Blog Microservices

> Building a blog website to learn how different services work together. We are having following 05 different services that are running on different ports.

| Service Name  | Port |
| ------------- | ---- |
| Posts         | 4000 |
| Comments      | 4001 |
| Query Service | 4002 |
| Moderation    | 4003 |
| Event Bus     | 4005 |

Every service is dockerized individually.
